# MotoFix Web

MotoFix is a web-based motorcycle service booking and customization system.

## Features

- User login and registration
- Customer dashboard
- Service booking system
- Admin, staff, mechanic, and customer roles
- Chat system
- Motorcycle parts customization
- AI motorcycle preview
- Supabase database integration

## Tech Stack

- React
- Vite
- Tailwind CSS
- Supabase

## How to Run

1. Install dependencies:

npm install

2. Run the project:

npm run dev

3. Build the project:

npm run build

## Environment Variables

Create a .env file inside the motofix-web folder.

Required variables:

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

Do not upload the .env file to GitHub.

## Important

The following files should not be uploaded to GitHub:

- node_modules
- dist
- .env
- .env.local

These should be included in .gitignore.
